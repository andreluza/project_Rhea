Este projeto tem as seguintes pastas:

"Arthur_shiny", com os códigos R e shiny para criar a webpage dedicada a coleta de informações de ornitologistas experts
"bugs", com alguns códigos em linguagem BUGS
"data", como todos os dados, desde deteccoes de Rhea em cada base de dados, covariaveis ambientais, dados espaciais (rasters, shapefiles), e também codigos R para extrair a informacao das diferentes fontes
"output", com os resultados dos modelos
"R", principais codigos de R necessarios para executar o projeto
"referencias_MS", algumas referencias e o texto do MS
